About MedExtract

What is MedExtract?

MedExtract is a clinical datapoint extraction system that uses Large Language Models (LLMs) and Retrieval-Augmented Generation (RAG) to extract specific information from medical reports.

Key Technologies

Large Language Models

Powered by state-of-the-art LLMs like Llama 3 and Mistral, MedExtract understands medical terminology and context to accurately extract clinical datapoints.

RAG Technology

Retrieval-Augmented Generation combines document retrieval with generation, ensuring that extracted information is grounded in the actual report content.

Vector Embeddings

Uses sentence transformers to create semantic embeddings of medical text, enabling intelligent retrieval of relevant information.

Performance Metrics

Comprehensive evaluation with accuracy, precision, recall, and F1 scores, along with confusion matrices for detailed analysis.

Target Audience

This dashboard is designed for:

  • Biologists and Researchers: Extract structured data from unstructured medical reports
  • Health Scientists: Analyze clinical outcomes and patient data at scale
  • Healthcare Professionals: Automate extraction of key clinical indicators
  • Data Analysts: Generate datasets for medical research and analysis

Features

  • Support for multiple LLM models (Llama 3, Mistral, Llama 2)
  • Configurable RAG with various embedding models
  • Multiple retriever types (vector store, ensemble)
  • Reranking for improved relevance
  • Few-shot learning capabilities
  • Rule-based fallback extraction
  • Comprehensive performance evaluation
  • Visual analytics with confusion matrices
  • Easy CSV export of results
